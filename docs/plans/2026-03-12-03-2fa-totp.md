# 两步验证 (TOTP/2FA) Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 为 DTax 添加 TOTP 两步验证，用户可通过 Google Authenticator / Authy 等应用生成一次性密码，提升加密资产税务平台的安全基线。

**Architecture:** 后端使用 `otpauth` 库生成 TOTP secret 和验证码。Secret 加密存储在数据库。前端展示 QR 码供用户扫描。登录流程增加 2FA 验证步骤。提供恢复码 (recovery codes) 作为备份。

**Tech Stack:** otpauth (TOTP), qrcode (QR生成), Prisma (schema), Fastify (API), Next.js (UI)

```typescript
// apps/api/src/lib/totp.ts
import * as OTPAuth from "otpauth";
import crypto from "crypto";

export function generateTotpSecret(email: string): {
  secret: string;
  uri: string;
} {
  const totp = new OTPAuth.TOTP({
    issuer: "DTax",
    label: email,
    algorithm: "SHA1",
    digits: 6,
    period: 30,
    secret: new OTPAuth.Secret({ size: 20 }),
  });

  return {
    secret: totp.secret.base32,
    uri: totp.toString(),
  };
}

export function verifyTotpToken(secret: string, token: string): boolean {
  const totp = new OTPAuth.TOTP({
    issuer: "DTax",
    algorithm: "SHA1",
    digits: 6,
    period: 30,
    secret: OTPAuth.Secret.fromBase32(secret),
  });

  const delta = totp.validate({ token, window: 1 });
  return delta !== null;
}

export function generateRecoveryCodes(count: number = 10): string[] {
  return Array.from({ length: count }, () => {
    const bytes = crypto.randomBytes(6);
    const hex = bytes.toString("hex");
    return `${hex.slice(0, 4)}-${hex.slice(4, 8)}-${hex.slice(8, 12)}`;
  });
}
```

### Task 6: Web API Client — 2FA 函数

**Files:**

**Step 1: 添加 2FA API 函数**

```typescript
export async function setup2FA() {
  return apiFetch<{ qrCodeUrl: string; secret: string }>("/auth/2fa/setup", {
    method: "POST",
  });
}

export async function verify2FA(token: string) {
  return apiFetch<{ enabled: boolean; recoveryCodes: string[] }>(
    "/auth/2fa/verify",
    {
      method: "POST",
      body: JSON.stringify({ token }),
    },
  );
}

export async function disable2FA(token: string) {
  return apiFetch<{ disabled: boolean }>("/auth/2fa/disable", {
    method: "POST",
    body: JSON.stringify({ token }),
  });
}

export async function get2FAStatus() {
  return apiFetch<{ enabled: boolean }>("/auth/2fa/status");
}

export async function loginWith2FA(
  tempToken: string,
  totpToken?: string,
  recoveryCode?: string,
) {
  return apiFetch<{ token: string; user: AuthUser }>("/auth/login/2fa", {
    method: "POST",
    body: JSON.stringify({ tempToken, totpToken, recoveryCode }),
    // 不需要 Authorization header — tempToken 在 body 中
  });
}
```

```typescript
const login = async (
  email: string,
  password: string,
): Promise<{ requiresTwoFactor?: boolean }> => {
  const res = await loginApi(email, password);
  if (res.requiresTwoFactor) {
    // 存储 tempToken，不设置完整 token
    setTempToken(res.tempToken);
    return { requiresTwoFactor: true };
  }
  // 正常登录流程
  setToken(res.token);
  setUser(res.user);
  return {};
};
```
