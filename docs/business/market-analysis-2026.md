## 三、监管驱动力（2026-2028 黄金窗口）

### 3.1 美国 IRS

| 时间线                       | 要求                                                  |
| ---------------------------- | ----------------------------------------------------- |
| **2025年报税（2026年申报）** | 券商报告总收入（gross proceeds），成本基础可选        |
| **2026年起**                 | 券商必须报告总收入 + 调整后成本基础（covered assets） |
| **DeFi 券商规则**            | 已被国会废除 → DeFi 用户必须自行报税                  |
| **Form 1040**                | 数字资产问题继续强制回答                              |

### 3.2 全球

| 监管框架      | 时间线                                  | 影响                         |
| ------------- | --------------------------------------- | ---------------------------- |
| **OECD CARF** | 2026数据采集，2027首次交换，75+国家承诺 | 巨大合规需求                 |
| **EU DAC8**   | 2026年实施                              | 所有EU成员国加密报告义务     |
| **日本**      | 20%分离课税改革推进中                   | 降税率→更多交易→更多报税需求 |
| **韩国**      | 再次推迟到2027年                        | 延期=合规准备窗口            |
| **印度**      | 30%+1% TDS 维持                         | 高税率+强执法→刚性需求       |
| **UK HMRC**   | CGT 报告+CARF承诺                       | Share Pooling 方法需求       |

### 3.3 关键结论

**2026年是监管拐点：** IRS成本基础报告生效、CARF数据采集开始、DAC8实施、MiCA全面执行。这是合规从"可选"变为"必须"的一年。dTax 在这个窗口期推出是最佳时机。

## 五、融资可行性分析

### 5.1 坦率评估：现阶段融资难度较高

**挑战：**

- 无营收（pre-revenue）—— 绝大多数VC要求至少有早期牵引力
- Solo founder —— VC 强烈偏好 2-3人创始团队
- 市场已有两个独角兽（CoinTracker $1.3B、TaxBit $1.33B）
- 2026年加密VC市场虽复苏但仍谨慎，集中在 L1/L2 和 DeFi 而非工具层

**有利因素：**

- 产品完成度极高（1400+测试、23解析器、8税务方法、7语言、AI）
- Open Core 差异化（无直接竞品）
- 2026监管拐点（CARF/DAC8/1099-DA）
- CPA $499/年定价 = 高LTV
- Koinly 证明了自举可行（~$6M ARR，极少融资）

### 5.3 融资路径选择

#### 路径 A：先自举，收入验证后再融资（推荐）

```
阶段1 (0-6个月): 发布 + 获取前100付费用户
  → 目标: $50K-$100K ARR
  → 验证: PMF (Product-Market Fit)

阶段2 (6-12个月): 增长到 $500K ARR
  → CPA渠道 + 自然增长 + 内容营销
  → 此时可寻求 Pre-Seed/Seed

阶段3 (12-24个月): Seed轮 $500K-$2M
  → 以 $500K+ ARR 和增长率为基础
  → 2026 SaaS Seed 基准: $300K-$1M ARR
```

**为什么推荐这条路：**

1. **Koinly 验证了自举路径** — 几乎无融资做到 $6M ARR、1M用户
2. **CoinTracker 在 $1.5M seed 后4年才拿 Series A** — 先证明模型再融资
3. **有收入的公司估值更高** — Pre-seed 无收入估值 $2-5M vs 有收入 $10-20M
4. **税务软件有天然的季节性牵引力** — Q1 报税季是最佳用户获取窗口

#### 路径 B：直接寻求 Pre-Seed（如需加速）

## 七、SWOT 总结

### Strengths（优势）

- 产品完成度极高（远超种子期平均水平）
- 唯一开源 Web 加密税务平台
- 8种税务方法 + 7语言 = 全球覆盖
- AI-first 架构（15+ providers）
- 502+ API 测试 = 工程质量高

### Weaknesses（劣势）

- Pre-revenue，无用户牵引力
- Solo founder
- 品牌知名度为零
- 无 CPA/税务行业人脉网络
- 无运营资金储备

### Opportunities（机会）

- **2026 监管拐点** — 全球合规从可选变必须
- **55-95% 合规缺口** — 巨大未开发 TAM
- **DeFi 券商规则废除** — DeFi 用户必须自行报税
- **CARF 2027** — 67 国强制报告
- **开源信任优势** — 税务计算透明度是护城河
- **CPA 渠道** — 一个 CPA = 多个客户

### Threats（威胁）

- CoinTracker/Koinly 已有网络效应
- TurboTax/H&R Block 可能内建加密税务
- 新竞品可能出现（Summ 品牌重塑）
- 加密熊市可能减少交易量/报税需求
- 监管延迟（如韩国反复推迟）
