dTax 使用您的**公开钱包地址**同步链上交易历史。从不需要也不会要求您的私钥或助记词。

<Aside type="tip" title="仅需公开地址">
dTax 只需要您的公开钱包地址 —— 以 `0x`、`bc1`、`1`、`3` 开头或 Solana base58 字符串。请勿将私钥或助记词分享给任何应用程序。
</Aside>

## 在 dTax 中输入地址的位置

1. 前往 **Settings** → **Wallets** → **Add Wallet**
2. 选择链（Ethereum、Solana、Bitcoin 等）
3. 粘贴您的公开地址
4. 点击 **Sync**

---

## EVM 链（Ethereum、Polygon、BSC、Arbitrum、Optimism 等）

dTax 支持 16 条 EVM 兼容链。同一个 Ethereum 地址（`0x...`）适用于所有链。

<Tabs>
<TabItem label="MetaMask">

<Steps>

1. **打开 MetaMask**

   点击浏览器中的 MetaMask 扩展图标。

2. **复制您的地址**

   地址显示在钱包顶部，以 `0x` 开头。点击即可复制。

   示例：`0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045`

3. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择 **Ethereum**（或具体链）→ 粘贴地址。

</Steps>

</TabItem>
<TabItem label="Coinbase Wallet">

<Steps>

1. **打开 Coinbase Wallet 应用**

   打开 Coinbase Wallet 移动应用或浏览器扩展。

2. **查找您的地址**

   点击 **Receive** 按钮或点击顶部的钱包名称。
   您的 Ethereum 地址（`0x...`）会显示出来。点击复制。

3. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择链 → 粘贴地址。

</Steps>

</TabItem>
<TabItem label="Ledger Live">

<Steps>

1. **打开 Ledger Live**

   打开 Ledger Live 桌面应用并连接您的 Ledger 设备。

2. **前往接收**

   点击左侧边栏的 **Receive** → 选择您的 Ethereum 账户。

3. **复制地址**

   您的地址会显示在屏幕上，同时也会在 Ledger 设备上显示以供验证。
   复制 `0x...` 地址。

4. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择链 → 粘贴地址。

</Steps>

</TabItem>
<TabItem label="Trust Wallet">

<Steps>

1. **打开 Trust Wallet**

   打开 Trust Wallet 移动应用。

2. **查找您的 ETH 地址**

   点击 **Receive** 按钮 → 选择 **Ethereum** → 您的 `0x...` 地址会显示出来。

3. **复制并粘贴**

   点击 **Copy** 复制地址。
   在 dTax → Settings → Wallets → Add Wallet → 选择链 → 粘贴地址。

</Steps>

</TabItem>
</Tabs>

<Aside type="note" title="一个地址，多条链">
您的 `0x` Ethereum 地址在 Polygon、BSC、Arbitrum、Optimism、Avalanche 及所有其他 EVM 链上都是相同的。您可以多次添加同一地址 —— 每条链各一次 —— dTax 会分别同步每条链的交易。
</Aside>

## Bitcoin

Bitcoin 有四种地址格式 —— dTax 全部支持。

| 格式 | 开头字符 | 示例 |
|------|----------|------|
| Legacy (P2PKH) | `1` | `1A1zP1eP5QGefi2DMPTfTL5SLmv7Divf...` |
| Nested SegWit (P2SH) | `3` | `3J98t1WpEZ73CNmQviecrnyiWrnqRhWNLy` |
| Native SegWit (Bech32) | `bc1q` | `bc1qar0srrr7xfkvy5l643lydnw9re59g...` |
| Taproot (Bech32m) | `bc1p` | `bc1p5cyxnuxmeuwuvkwfem96l...` |

<Tabs>
<TabItem label="Bitcoin Core">

<Steps>

1. **打开 Bitcoin Core**

2. **前往接收**

   点击 **Receive** 标签页。

3. **复制您的地址**

   显示一个收款地址。点击 **Copy Address**。

4. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择 **Bitcoin** → 粘贴地址。

</Steps>

</TabItem>
<TabItem label="Electrum">

<Steps>

1. **打开 Electrum**

2. **查看您的地址**

   前往 **View** → **Show Addresses**（如果不可见请启用）。
   您的收款地址列在 **Receiving** 标签页中。

3. **复制地址**

   右键点击任意地址 → **Copy to clipboard**。

4. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择 **Bitcoin** → 粘贴地址。

</Steps>

<Aside type="note" title="HD 钱包与 xpub">
Electrum 为每笔交易生成新地址（HD 钱包）。dTax 支持通过单个 xpub/ypub/zpub 密钥**一次性同步所有地址** —— 无需逐一添加每个地址。在 Electrum 中，前往 **Wallet** → **Information** 可找到您的主公钥。
</Aside>

</TabItem>
<TabItem label="Ledger Live (Bitcoin)">

<Steps>

1. **打开 Ledger Live**

   连接您的 Ledger 设备并打开 Ledger Live。

2. **前往接收**

   点击 **Receive** → 选择您的 Bitcoin 账户。

3. **复制地址**

   Ledger 显示一个新的收款地址。点击 **Copy**。

4. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择 **Bitcoin** → 粘贴地址。

</Steps>

</TabItem>
<TabItem label="Exodus">

<Steps>

1. **打开 Exodus**

2. **找到 Bitcoin 钱包**

   从钱包列表中选择 **Bitcoin**。

3. **点击接收**

   点击 **Receive** 按钮。您的 Bitcoin 地址以二维码和文本形式显示。
   点击 **Copy** 复制。

4. **粘贴到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择 **Bitcoin** → 粘贴地址。

</Steps>

</TabItem>
</Tabs>

## HD 钱包（xpub / ypub / zpub）

如果您使用硬件钱包（Ledger、Trezor）或 Electrum 管理 Bitcoin，可以使用扩展公钥**一次性同步所有派生地址**。

| 密钥格式 | 地址类型 |
|----------|----------|
| `xpub...` | Legacy (P2PKH) |
| `ypub...` | Nested SegWit (P2SH) |
| `zpub...` | Native SegWit (Bech32) |

<Steps>

1. **查找您的扩展公钥**

   - **Ledger Live**：前往 Bitcoin 账户 → **...** 菜单 → **Advanced** → **Extended public key (xpub)**
   - **Trezor Suite**：前往 Bitcoin 账户 → **...** → **Public key**
   - **Electrum**：**Wallet** → **Information** → Master Public Key

2. **添加到 dTax**

   在 dTax → Settings → Wallets → Add Wallet → 选择 **Bitcoin** → 粘贴 xpub/ypub/zpub。

   dTax 会自动派生所有子地址并同步每个地址的交易。

</Steps>

<Aside type="caution" title="xpub 会暴露您所有的地址">
您的 xpub 不是私钥 —— 它无法转移资金。但是，它会向任何持有它的人暴露您所有的 Bitcoin 地址和余额。请仅与您信任的服务共享。
</Aside>

## 故障排除

**同步后"No transactions found"**
- 确保粘贴了正确的地址（检查首尾几个字符）。
- 验证是否选择了正确的链 —— 错误链上的 ETH 地址（例如选了 Arbitrum 而非 Ethereum）将不显示结果。
- 部分钱包的交易数量本就很少 —— 这是正常的。

**交易记录缺失或不完整**
- dTax 从最早可用的区块开始同步。对于非常旧的钱包（2018 年以前），部分索引器的历史数据可能有限。
- 尝试重新添加同一地址 —— 它将从 API 重新获取数据。

**交易数量有误**
- 如果您在多条链上使用了相同地址（EVM 常见情况），请为每条链分别添加。

**Bitcoin：只有部分地址显示交易**
- 如果您使用 HD 钱包，请添加 xpub 密钥而非单个地址 —— 这样可以自动覆盖所有派生地址。
